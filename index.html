O que é o WordPress?

O WordPress é um projeto de código aberto (Open Source) que você pode usar para criar sites, blogs ou aplicativos.

Quem usa WordPress?

35% da internet usa o WordPress, de blogs amadores até os maiores sites de notícias. Sony, Walt Disney Company, Toyota e vários outras empresas e negócios ao redor do mundo.

Mas o que é Gutenberg e o que muda no WP?

Gutenberg foi o nome dado ao novo editor do WordPress. A maneira de editar foi inteiramente reconstruída para páginas e posts com mídia rica. Surgiram os blocos, que trazem facilidade e flexibilidade impressionante na hora de criar post, página e demais conteúdos para o WordPress.

Como faço para usar o Gutenberg?

O Gutenberg está disponível como parte do WordPress 5.0 e posteriores. Através do novo editor ficou mais fácil a criação de conteúdos engajadores. Com os blocos do Gutenberg você pode inserir, rearrumar e dar estilo para os conteúdos multimídias com muito pouco conhecimento técnico. Em vez de usar um código personalizado, você pode adicionar um bloco e se concentrar no conteúdo.

O Gutenberg é para usuários comuns ou só para programadores?

O novo editor Gutenberg pode ser utilizado por ambos. Ele está mudando a forma como utilizamos o WordPress, tornando tudo mais simples e intuitivo, tanto para usuários como para desenvolvedores.
